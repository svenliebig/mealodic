# Application

This document explains the application structure on a higher level.

## Unauthenticated Users

Unauthenticated users will see a modern welcome landing page, they can also browse public recipes and have the possibility to signup or login.

## Authenticated Users

Authenticated users will see the main application layout, they can browse recipes, meal plans, shopping lists, ingredients, families, etc.

## Admin Users

Admin users will see the admin panel, they can manage users, recipes, meal plans, shopping lists, ingredients, families, etc.
