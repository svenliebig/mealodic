# Families

Families are groups of users that can share recipes, meal plans and shopping lists. A user can have multiple families.

## Creating a Family

The user can create a family by clicking on the create family button in the families section. The user will be asked to provide a name for the family. The family will be created and the user will be added to the family.

## Adding Users to a Family

The user can add users to a family by clicking on the add user button in the family details. The user will be asked to provide the email address of the user they want to add. The added user will be added as pending member to the family. The added user will have a pending notification in their notifications section, the user can click on the bell icon in their UI and accept the invitation to join the family.

## Removing Users from a Family

The user can remove users from a family by clicking on the remove user button in the family details next to the family member. The user will be removed from the family and will no longer be able to access the family's recipes, meal plans and shopping lists.

## Ownership

If an owner leaves the family he needs to nominate a new owner. If the user is the last owner of the family, the family, all it's meal plans and shopping lists will be deleted.
