# Families

Families are groups of users that can share recipes, meal plans and shopping lists.

## Creating a Family

The user can create a family by clicking on the create family button in the families section. The user will be asked to provide a name for the family. The family will be created and the user will be added to the family.

## Adding Users to a Family

The user can add users to a family by clicking on the add user button in the family details. The user will be asked to provide the email address of the user they want to add. The user will be added to the family and will receive an email with a link to join the family.

## Removing Users from a Family

The user can remove users from a family by clicking on the remove user button in the family details. The user will be asked to provide the email address of the user they want to remove. The user will be removed from the family and will no longer be able to access the family's recipes, meal plans and shopping lists.
